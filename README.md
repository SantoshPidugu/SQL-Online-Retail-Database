Online Retail Store SQL Database Project

This project demonstrates the design and implementation of a synthetic multi-table SQL database using SQLite.

Features

6 relational tables

1500+ transaction records

Composite primary key (Order_Items)

Foreign key relationships

Deliberate missing and duplicate data

Realistic statistical data distributions

Technologies Used

Python

SQLite

Faker

NumPy

The database was generated programmatically and is designed for customer behaviour and retail analytics simulations.
